H5YCK v1.3

HTML5 YAML Compass (SASS) Kickstarter

v1.3.1

This is my construction set for starting a new web project. Basically it contains elements from the HTML5 Boilerplate, the CSS Framework YAML, the CSS preprocessor SASS in combination with the SASS Framework Compass and the Build Script to compress the result.

Also the JavaScript is written in CoffeeScript but all CoffeeScript files are compiled to readable JavaScript (no compression). So you don’t need to know CoffeeScript and can work with the compiled JavaScript files.

Dependencies

  • Node.js
  • Grunt.js
  • SASS
  • Compass
  • Phantom.js

How to use

Clone the project

git clone https://github.com/michsch/h5yck.git

Move into the project folder and fetch your dependencies

cd h5yck
npm install
sudo npm install grunt-compass
sudo npm install grunt-contrib

Start grunt

grunt
